# NIKE_vs_ADIDAS
PROJECT EXPLAINATION

Title of my project is "Classifying a shoe as NIke or Adidas using CNN."

The main objective of this project is classifying a given shoe imae using Keras in Google Colab to see if its
adidas or nike shoe.

In this project
I have build a CNN model with multiple convolution, pooling layers to extract image features.
The model was trained on the shoe image data set and validated against test images.
I have used RELU and softmax activation functions and Adam optimizer for better performance in this project.
I have also used KERAs framework to build the neurol network in this project.
